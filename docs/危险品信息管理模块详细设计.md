# 危险品信息管理模块详细设计

## 1. 模块概述

危险品信息管理模块是危险品物流管理系统中的核心模块之一，主要用于管理系统中所有的危险品信息，包括危险品的基本信息、类型分类、规格参数、存储数量、注意事项等。该模块为运输调度、车辆分配、应急处置等模块提供危险品数据支撑。
